# KeyForge

KeyForge 是一个交互式训练工具，帮助你学习和掌握 VSCode 快捷键。 

> 这个插件是完全基于 Google AntiGravity 的 Vibe coding 结果，我本人并不会 TypeJs 开发。产生做这个插件的想法是因为我尝试使用 AntiGravity 时从 vscode 迁移配置和快捷键到 AntiGravity，结果发现 AntiGravity 的快捷键配置和 vscode 的快捷键有很大差异，仔细查看才知道是我的 vscode 的快捷键一部分是从 IntelliJ IDEA 迁移过来的，一部分又是因为我使用了 vim 插件，所以就感觉使用体验打了折扣。毕竟现在套壳 vscode 的 AI IDE 层出不穷，为了标准化的快捷键，我需要尽快训练自己重新适应 vscode 标准快捷键，因此做了这个插件来形成快捷键的肌肉记忆。
